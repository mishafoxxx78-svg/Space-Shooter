## 🎮 Описание проекта

**Space Shooter** — это 2D аркадная игра, созданная на Python с использованием библиотеки Pygame.

Игрок управляет космическим кораблём, уничтожает врагов, набирает очки и пытается выжить как можно дольше.

Проект демонстрирует навыки:

- Работа с игровым циклом
- Обработка событий
- ООП
- Коллизии (столкновения)
- Работа с графикой
- Управление состоянием игры
